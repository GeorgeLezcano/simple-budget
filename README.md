Simple Budget is a lightweight Windows desktop app for tracking **income**, **expenses**, and basic **budget summaries** — with a clean, dark-themed UI and a simple workflow.

This repo is the start of a personal finance tracker focused on **fast entry**, **clear monthly/weekly/yearly views**, and **exportable reports**.

---

## What it does

- Track **Income** and **Expenses**
- Show a **Dashboard** summary (Income / Expenses / Savings / Net)
- Use an **anchor date** + range selector (**Week / Month / Year**) to view totals
- Manage **Income** and **Expense** categories in **Settings**
- Store a **Savings %** setting used to compute “Savings” totals
- Filter data in **Reports** and export results

> Note: Some features are still being built and may not be fully functionally yet.

---

## Requirements

- Windows 10+
- .NET 8 (Windows Forms) project if building the app locally.
- Write permission of the directory where the **.exe** is located.
